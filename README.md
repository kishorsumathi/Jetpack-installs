## Jetpack-installs

Howto's for building, making, and installing packages for jetpack.


### Python

[Install python 3.9.12 or other python version with altinstall](https://www.google.com)

This will show you how to altinstall a python version, so to not replace your default Python3 that came with jetpack.

### openCV 4.5.5
[Install openCV 4.5.5 or other openCV version](https://www.google.com)

This is for installing openCV to a altinstall of python or your default Python3 that came with jetpack.
